Introduction

Welcome to the baserowapi documentation. baserowapi is a high-level API wrapper designed to integrate with Baserow (https://baserow.io), the open-source no-code database tool.

Getting Started

The Quick Start is available for those eager to set up and initiate their first API call with baserowapi.

Feedback and Contributions

Feedback is appreciated. Whether it’s a feature request, a bug report, or comments on this documentation, input is welcome. Please fork the repository and create a pull request for any proposed changes.
